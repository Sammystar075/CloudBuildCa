# Kindle Suite v1.0

A collection of web apps optimized for Kindle's e-ink browser.

## Included Apps

- **Markdown Editor** - Write and preview markdown documents
- **Kindoku** - Sudoku puzzles with 3 difficulty levels  
- **Minesweeper** - Classic mine-finding game
- **Picross** - Picture logic puzzles

## Installation

1. Download `kindle-suite-v1.0.zip`
2. Extract the zip file
3. Connect your Kindle via USB
4. Drag `documents/` contents → Kindle's `documents/` folder
5. Drag `extensions/` contents → Kindle's `extensions/` folder
6. Safely eject Kindle
7. Open "Kindle Suite" from your library

## Requirements

- Jailbroken Kindle with shell script execution
- Kindle's experimental browser enabled

## Features

- E-ink optimized (high contrast, no animations)
- Works offline
- Touch-friendly controls
- Kindle browser compatible (ES5, no flexbox)
